# OA办公管理系统 java500

## 项目概述

OA办公管理系统是基于Java语言的SSM框架开发的B/S架构的Web应用。该系统采用MVC三层设计模式，集成了Spring、SpringMVC、Mybatis等后端技术，以及Jsp、CSS、Html、JavaScript等前端技术，提供了一套完整的OA办公解决方案。

## 技术栈

### 后端技术

- Java
- MySQL
- Spring
- SpringMVC
- Mybatis

### 前端技术

- Jsp
- CSS
- Html
- JavaScript

## 系统角色

系统包含但不限于以下角色：

- 系统管理员
- 普通员工
- 部门经理

## 运行环境

- 系统环境：Windows
- 开发环境：JDK 1.8、Maven、MySQL、Tomcat 9.0
- 数据库工具：Navicat 11
- 开发软件：Eclipse/MyEclipse/IDEA
- Maven版本：Maven 3.3.9
- 推荐浏览器：谷歌浏览器

## 部署步骤

1. 配置Java开发环境（JDK 1.8）
2. 安装并配置Maven（3.3.9）
3. 安装并配置MySQL数据库（5.7）
4. 安装并配置Tomcat服务器（9.0）
5. 导入数据库文件
6. 部署应用到Tomcat
7. 启动应用，访问系统
